Java笔记

发表于 2019-01-02

记录一些Java特性的知识点，便于复习。

this关键字

  • 一般使用

对象方法中代表对象自身。

  • 内部类中的this

可以指定OuterClass.this明确指定外部类对象引用。

static关键字

  • 修饰属性

static关键字修饰的属性是类属性，可以通过类访问，也可以通过对象访问，所有对象共享这一个属性。

  • 修饰方法

static关键字修饰的方法是类方法，可以通过类访问，也可以通过对象访问。

  • 修饰内部类

静态内部类。

final关键字

  1. 声明一个常量或者常量属性。

  2. 修饰方法，该方法能被继承，但是不能被重写。

  3. 修饰类，该类不能被继承。

  4. 在内部类中使用局部变量时给变量添加final关键字 。final定义的变量会放在常量池，生命周期从局部变成全局。可以确保内部类不会访问到已经释放的参数，导致野指针访问的错误。

代码块

代码块是用{}包起来的一块代码，有以下几种：

  • 普通代码块

方法中的代码块，起到作用域的作用。

  • 构造代码块

类定义中的代码块，每次生成对象的时候会调用。在构造方法之前调用。

  • 静态代码块

类定义中的代码块，有static关键字修饰。第一次使用类的时候会调用，只执行一次。

继承

  1. 不继承构造方法。

  2. 不继承static方法。

  3. 不继承final方法。

  4. 不继承private方法。

方法重载

方法名和返回值相同，参数列表不同。

方法重写

子类重写继承的方法，需要注意以下几点：

  1. 重写方法抛出的异常不能是父类方法抛出的异常的父类。

  2. 重写方法的访问级别不能低于父类方法的访问级别。

  3. private，static，final修饰的方法不能被重写，因为这几个关键字修饰的方法是不能被继承的。

抽象类

抽象类作为类地抽象，有以下几个特点：

  1. 抽象类不能实例化。

  2. 抽象方法必须定义在抽象类中。

  3. 抽象类不一定含有抽象方法。

接口

  1. 接口中可以定义静态常量和方法。

  2. 静态常量默认是public static final修饰。

  3. 方法默认是public abstract修饰。

  4. 实现类必须实现协议方法，抽象类可以不实现协议方法。

  5. JDK1.8之后可以提供方法的默认实现，方法名前加default关键字。

内部类

内部类分为以下几种:

  • 成员内部类

会和外部类对象绑定，内部类对象可以访问外部类对象的成员和方法。

  • 静态内部类

不会和外部类对象绑定，内部类可以访问外部类的静态成员和静态方法。

  • 方法内部类

会和外部类对象绑定，内部类对象可以访问外部类对象的成员和方法。

  • 匿名内部类

会和外部类对象绑定，内部类对象可以访问外部类对象的成员和方法。

为什么需要使用内部类？

  1. 可以实现多继承。

  2. 可以更好地隐藏内部实现。比如遵循某个协议，但是并不想让外部知道时。
