使用hexo搭建个人博客

发表于 2019-01-02 | 更新于 2019-01-21

hexo是一个快速，简洁且高效的博客框架，基于Node.js开发。使用它我们可以快速地搭建个人博客。

hexo官方网站

前提

在安装hexo前需要确保已经安装以下程序：

  • Git
  • Node.js

安装hexo

npm是Node.js平台的默认包管理工具（类似iOS开发中的CocoaPods）。-g参数表示全局安装，工具会被安装在全局目录下，可以在任何地方使用。

初始化

创建一个文件夹开始搭建博客。

进入到刚才创建的文件夹。

执行hexo初始化命令。

安装hexo依赖，npm install会根据当前目录下的package.json配置文件安装依赖组件。和CocoaPods中的pod install命令会根据Podfile安装依赖库是一样的意思。

完成初始化后，目录结构如下：

_config.yml

配置文件，可以在此配置大部分的参数。

package.json

hexo依赖配置文件。

scaffolds

模版文件夹。当新建文章时，Hexo 会根据 scaffold 来建立文件。

source

资源文件夹是存放用户资源的地方。

themes

主题文件夹。Hexo 会根据主题来生成静态页面。

生成博客

使用 hexo g 命令生成博客的静态网页。

写作

使用 hexo new 命令新建一篇文章。

执行命令后会在 source/_posts 目录下生成 title.md 文件。我们可以在这个文件中使用 Markdown 语法编写文章内容。完成之后再执行 hexo g 命令重新生成静态网页，启动本地服务查看效果。
